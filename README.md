# Hey👋 Sejam bem-vindos!

## Sobre mim

Me chamo Marcelo Lourenço e sou Desenvolvedor Front-end. Atualmente, estou cursando Engenharia de Software (1º semestre) na Universidade de Mogi das Cruzes, além de realizar cursos online focados na minha área de atuação para complementar minha formação.

Desde sempre, sou apaixonado por tecnologia, e essa paixão me motivou a iniciar meus estudos em Lógica de Programação, Arquitetura de Software, entre outros tópicos essenciais. Essa jornada construiu uma base sólida de conhecimentos, o que me permite aprender novas linguagens de programação de forma ágil e eficaz. Estou sempre em busca de novos desafios para expandir minhas habilidades e contribuir para o desenvolvimento de soluções inovadoras.

Tenho uma grande paixão por criar soluções que priorizem acessibilidade, agilidade e facilidade para o usuário, proporcionando uma experiência acolhedora, tendo em vista minhas habilidades e o impacto positivo na experiência do usuário ao acessar meu projeto.

No meu ambiente de estudos, tenho a oportunidade de desenvolver projetos pessoais com foco em JavaScript e Node.js. Tenho experiência no consumo de APIs e na interação com bancos de dados como MongoDB e PostgreSQL, sempre buscando otimizar a performance e garantir a integridade das informações. Utilizo frameworks como Next.js e Express.js para criar aplicações escaláveis e eficientes. Para gerenciar e versionar meu código, faço uso de Git e GitHub, o que me permite colaborar com a comunidade de desenvolvedores, além de aprender e trocar conhecimentos com outros profissionais. Com isso, procuro sempre aprimorar minhas habilidades e contribuir para o crescimento do ecossistema tecnológico.
